Welkom op AmiTwitter
veelgestelde vragen

Wat betekent AmiTwitter voor staan?

Antwoord: A miga- M orphOS- I KE-Twitter

Wat zijn de vereisten voor AmiTwitter?

Antwoord: MUI, htmltext, urlText, TheBar, BetterString, codesets.library, libiconv.ixlibrary. OpenURL is optioneel.

AmiTwitter niet downloaden of versturen Tweets?

Antwoord: Bent u aangesloten op het internet, heb je je opgegeven Twitter gebruikersnaam / wachtwoord in het Tools -> Settings-venster en heeft hen gered? Bent u al na enkele mensen (voor Timeline om te werken) en heb je gestuurd sommige Tweets (voor Mijn Tweets om te werken)?

Directe Berichten lijken niet te werken?

Antwoord: Zorg ervoor dat u Screen Naam van de ontvanger (dwz User Name) correct, is het momenteel niet accepteren user id-nummers.

Ik houd AmiTwitter, hoe kan ik doneren?

Antwoord: Volg de 'Doneer' link naar de website van PayPal, dat opent in uw favoriete webbrowser

Er gebeurt niets als ik klik op de links op de bodem van de belangrijkste programmavenster (OS3.x)?

Antwoord: Zorg ervoor OpenURL correct is geïnstalleerd / geconfigureerd.

Alles wat ik zie is de boodschap, "proberen om de nieuwste ..."

Antwoord: Meest waarschijnlijke reden is hoeft u niets in die bepaalde tijdlijn. Zo heb je nooit 'favoriet' geen producten op de Twitter site, en zullen daarom niet worden weergegeven onder 'Mijn favorieten' in AmiTwitter niet.

Waarom is de bestandsgrootte zo groot?

Antwoord: Omdat AmiTwitter beroept zich op tal van bibliotheken om te werken, te weten: cURL, SSL, xml2, glib, enz. ...

Waarom schrijf je AmiTwitter?

Antwoord: Om te bewijzen dat het kan worden gedaan!

Wat is AmiTwitter gebaseerd?

Antwoord: xTwitter door Tsukasa Hamano

Links in de belangrijkste Tweets venster niet werken?

Antwoord: Dit is niet een webbrowser ... Ook ... het is een beperking van HTMLtext.mcc momenteel gebruikt

Wat zijn de toekomstplannen voor AmiTwitter?

Antwoord: Ten eerste, leer meer van de Twitter API, maar het hangt sterk af van de feedback die ik ontvang. Ook zou poorten naar andere systemen (dwz AROS) en een actieve bron code base met enkele ontwikkelaars toetreding tot het project op SourceForge leuk zijn ...
